# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - 2026-03-24

- Initial public release line for easySpectra.
- Local-first CLI + FLTK GUI pipeline from structure input to simulated spectra.
- Added workflow modularity with `all`, `nmr`, and `cd` product paths and shared manifest outputs.
- Expanded NMR support and comparison workflow, including `31P` support and improved overlay handling.
- Added experimental spectrum loading/switching support and curated easy/medium/hard example packs for validation.
- Improved GUI usability: cleaner peak highlighting, export snapshots, and general control-bar/layout polish.
- Added recurring checklist/test docs for experimental conversion and regression checks.
- Renamed project-facing branding to `easySpectra` while keeping existing CLI binary compatibility.
