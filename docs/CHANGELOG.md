# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1-alpha.0] - 2026-03-24

- Initial public pre-alpha repository bootstrap.
- Local-first CLI + FLTK GUI pipeline from structure input to simulated NMR.
- Multi-nucleus simulation path with `1H`, `13C`, and `19F` (when present).
- Atom-to-peak linking, spectrum zoom selection, workflow monitor, and audit artifacts.
